# ATM Interface in Java

## Overview

Welcome to the ATM Interface project in Java! This project simulates a basic ATM system, allowing users to perform key banking operations such as depositing funds, withdrawing cash, checking account balances, and transferring money to other bank users.

## Features

1. **Deposit Amount:**
   - Users can deposit funds into their accounts.
   - A straightforward feature for increasing account balance.

2. **Withdraw Amount:**
   - Users can withdraw cash from their accounts.
   - Provides financial flexibility for users.

3. **Check Balance:**
   - Users can inquire about their account balance.
   - A simple yet crucial feature for account management.

4. **Transfer Amount:**
   - Users can transfer funds to other bank users.
   - Fosters a connected and convenient banking environment.

## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Satyaki0007/ATM_Interface.git
   cd ATM_Interface
   ```

2. **Compile and Run:**
   ```bash
   javac ATMInterface.java
   java ATMInterface
   ```

3. **Follow On-Screen Instructions:**
   - The program will guide you through the ATM operations.
   - Enter options to deposit, withdraw, check balance, or transfer funds.
